U Terms

UDP (User Datagram Protocol)

User Datagram Protocol (UDP) is a connectionless network protocol that enables fast, lightweight data transmission without guaranteeing delivery or order. UDP is used in applications where speed is critical, such as streaming, gaming, and VoIP.

UHF (Ultra High Frequency)

Ultra High Frequency (UHF) refers to radio frequencies between 300 MHz and 3 GHz, used for television broadcasting, mobile phones, Wi-Fi, and satellite communication.

Uninterruptible Power Supply (UPS)

An Uninterruptible Power Supply (UPS) is a backup power device that provides emergency electricity to computers and equipment during power outages, preventing data loss and hardware damage.

Unicode

Unicode is a universal character encoding standard that assigns unique codes to every character in every language. Unicode enables consistent text representation and processing across platforms and applications.

Universal Serial Bus (USB)

Universal Serial Bus (USB) is a standard for connecting peripherals to computers and devices. USB supports data transfer, power delivery, and device charging, and is ubiquitous in modern electronics.

URL (Uniform Resource Locator)

A Uniform Resource Locator (URL) is the address used to access resources on the internet, such as web pages, images, and files. URLs are essential for navigation and linking in the World Wide Web.

Uptime

Uptime is the amount of time a system, server, or network has been continuously operational without interruption. High uptime is a key metric for reliability and service quality.

User Interface (UI)

A User Interface (UI) is the point of interaction between a user and a computer system, application, or device. UI design focuses on usability, accessibility, and user experience.

UUID (Universally Unique Identifier)

A Universally Unique Identifier (UUID) is a 128-bit number used to uniquely identify information in computer systems. UUIDs are used in databases, distributed systems, and software development.

Utility Software

Utility software is a category of system software designed to help manage, maintain, and optimize computer resources. Examples include antivirus programs, disk cleaners, and backup tools.
