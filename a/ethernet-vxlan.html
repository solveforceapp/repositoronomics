Ethernet VXLAN

Definition

Ethernet VXLAN (Virtual Extensible LAN) is a network virtualization technology that encapsulates Ethernet frames within UDP packets, enabling scalable Layer 2 networks across Layer 3 infrastructure.

Application

Used in data centers and cloud environments to support large-scale, multi-tenant networks and seamless workload mobility.
