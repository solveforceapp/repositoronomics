C Terms

Cache Coherency

Cache coherency is the process of keeping data in multiple caches synchronized in a multiprocessor shared memory system. When one processor updates a memory location, all other caches must reflect that change to maintain data consistency. Coherency protocols, such as MESI, are essential for correct operation in modern multicore CPUs and distributed systems.

Cache Eviction

Cache eviction is the process of removing data from a cache to make room for new entries. Eviction policies, such as Least Recently Used (LRU) or First-In-First-Out (FIFO), determine which data is discarded. Effective cache eviction improves performance by maximizing cache hit rates and minimizing memory access latency.

Cache Hit

A cache hit occurs when requested data is found in a local cache, avoiding the need to access slower main memory or storage. High cache hit rates are critical for system performance, especially in CPUs, databases, and web applications.

Cache Line

A cache line is a small block of memory within a cache, typically 32â€“128 bytes in size. Data is transferred between main memory and cache in cache line units, optimizing memory access patterns and reducing latency.

Cache Miss

A cache miss occurs when requested data is not found in the cache, requiring retrieval from a slower memory tier. Cache misses can degrade performance, making cache design and management crucial in computing systems.

Cache Thrashing

Cache thrashing is a pathological situation where frequent cache misses occur due to poor access patterns, causing data to be repeatedly evicted and reloaded. Thrashing reduces system efficiency and can be mitigated by optimizing algorithms and cache policies.

Cache Ways

Cache ways refer to the number of potential cache lines in an associative cache that specific physical addresses can be mapped to. Higher associativity reduces conflict misses but increases complexity and power consumption.

Caddy

A caddy is a tray or enclosure used to hold removable drives, such as hard disks or optical drives, in a computer or server. Caddies simplify drive installation, replacement, and hot-swapping in storage systems.

Calibration

Calibration is the process of adjusting hardware or software to ensure accuracy and reliability. In computing, calibration may involve tuning sensors, displays, or input devices to meet specified standards.

Callback

A callback is a function or executable code passed as an argument to other code, expected to be executed at a specific time or event. Callbacks are fundamental in event-driven programming, asynchronous operations, and user interface design.

CardBus

CardBus is a 32-bit version of the PC Card (PCMCIA) standard, providing faster data transfer rates for laptop expansion cards. CardBus supports devices such as network adapters, storage, and modems, and was widely used before the advent of USB and ExpressCard.

Card Punch

A card punch is a machine that creates holes in punched cards to store data or instructions. Card punches were essential in early computing for programming and data entry before the development of modern storage media.

Card Reader

A card reader is a device that reads data from card-shaped storage media, such as memory cards, smart cards, or magnetic stripe cards. Card readers are used in computers, ATMs, access control systems, and payment terminals.

Case Badge

A case badge is a small label or emblem affixed to a computer case, often displaying the manufacturer, model, or branding. Case badges are used for identification, marketing, and customization.

Case Fan

A case fan is a cooling device installed inside a computer case to improve airflow and dissipate heat from components. Proper cooling is essential for system stability, performance, and longevity.

Cassette

A cassette is a magnetic tape storage format used for audio, video, and data recording. In computing, cassettes were used for data storage in early personal computers and gaming consoles.

Central Processing Unit (CPU)

The Central Processing Unit (CPU) is the primary component of a computer that executes instructions, performs calculations, and manages data flow. Modern CPUs contain multiple cores, caches, and support for parallel processing, enabling high-performance computing across a wide range of applications.

Channel I/O

Channel I/O is a high-performance input/output architecture that offloads data transfer tasks from the CPU to dedicated channel processors. Channel I/O is used in mainframes and high-end servers to improve throughput and efficiency.

Character

A character is a unit of information representing a symbol, letter, or digit in a natural language. Characters are encoded using standards such as ASCII or Unicode, enabling text processing and communication in software systems.

Chip

A chip, or integrated circuit, is a small piece of semiconductor material containing electronic components such as transistors, resistors, and capacitors. Chips are the building blocks of modern electronics, powering everything from computers to smartphones.

Chipset

A chipset is a group of integrated circuits designed to work together, managing data flow between the CPU, memory, storage, and peripherals. Chipsets determine system capabilities, compatibility, and performance.

CI/CD

CI/CD stands for Continuous Integration and Continuous Delivery (or Deployment), a set of practices in software development that automate building, testing, and deploying code. CI/CD pipelines improve software quality, speed up releases, and enable rapid iteration.

CISC

Complex Instruction Set Computing (CISC) is a CPU architecture that supports a wide range of instructions, allowing complex operations to be performed with a single instruction. CISC architectures, such as x86, are common in desktop and server processors.

Class

A class is a blueprint for creating objects in object-oriented programming. Classes define attributes (data) and methods (functions) that describe the behavior and state of objects. Classes enable code reuse, encapsulation, and abstraction.

Class-Based Programming (Class-Orientation)

Class-based programming is a style of object-oriented programming where inheritance and behavior are defined via classes, rather than objects alone. This approach is used in languages like Java, C++, and Python.

Cleanroom Software Engineering

Cleanroom software engineering is a development process focused on defect prevention rather than removal. It emphasizes formal methods, incremental development, and statistical quality control to produce highly reliable software.

Client

A client is a piece of hardware or software that accesses services provided by a server, often over a network. Clients include web browsers, email programs, and mobile apps, enabling user interaction with remote resources.

Clone

A clone is a non-original copy of hardware or software, often produced by third parties. Clones may offer compatibility with popular products at lower cost, but may lack support or quality assurance.

Closed Architecture

Closed architecture refers to proprietary hardware or software designs that restrict modification, expansion, or interoperability. Closed systems may offer security and stability but limit user customization and third-party development.

CMOS

Complementary Metal-Oxide-Semiconductor (CMOS) is a technology used to construct integrated circuits, including microprocessors, memory, and sensors. CMOS chips are known for low power consumption and high density.

Code Library

A code library is a collection of pre-written code, functions, or classes that can be reused in multiple programs. Libraries accelerate development, promote best practices, and reduce duplication.

Code Refactoring

Code refactoring is the process of restructuring existing code to improve readability, maintainability, and performance without changing its external behavior. Refactoring helps reduce technical debt and enhance software quality.

Coding Theory

Coding theory studies the properties of codes for applications such as data compression, cryptography, and error detection/correction. It provides mathematical foundations for reliable data transmission and storage.

Cognitive Science

Cognitive science is the interdisciplinary study of the mind and its processes, including perception, memory, reasoning, and language. It combines psychology, neuroscience, computer science, linguistics, and philosophy to understand intelligence and cognition.

Collection (Abstract Data Type)

A collection is an abstract data type that groups multiple data items with shared significance, allowing operations such as addition, removal, and iteration. Collections include lists, sets, queues, and stacks.

Comma-Separated Values (CSV)

Comma-Separated Values (CSV) is a plain text file format for storing tabular data, where each line represents a row and values are separated by commas. CSV files are widely used for data exchange, import/export, and spreadsheet applications.

Command

A command is an instruction given to a computer to perform a specific operation. Commands are used in command-line interfaces, scripting, and automation to control software and hardware.

Command Line

The command line is a text-based interface where users type commands to interact with the operating system or software. Command-line interfaces (CLIs) offer powerful control and automation capabilities for advanced users.

Command Prompt

The command prompt is the text displayed in a command-line interface before the cursor, indicating readiness to accept input. It often shows information such as the current directory or user.

Comment

A comment is a piece of text in source code that is ignored by the compiler or interpreter. Comments are used to document code, explain logic, and improve readability for developers.

Compact Disc-Recordable (CD-R)

CD-R is an optical disc format that can be written to once, allowing users to store data, music, or video. CD-Rs are widely used for backups, distribution, and archival purposes.

Compact Disc-ReWritable (CD-RW)

CD-RW is an optical disc format that can be written to, erased, and rewritten multiple times. CD-RWs offer flexibility for data storage and transfer, but have lower compatibility than CD-Rs.

Compact Disc Read-Only Memory (CD-ROM)

CD-ROM is a pre-pressed optical disc containing data or music that cannot be modified. CD-ROMs are used for software distribution, multimedia, and reference materials.

CompactFlash

CompactFlash is a flash memory card format used in digital cameras, industrial equipment, and embedded systems. CompactFlash cards offer high capacity, durability, and speed.

Compiler

A compiler is a program that translates source code written in a high-level programming language into machine code or intermediate code. Compilers enable software to run efficiently on target hardware platforms.

Compiled Language

A compiled language is a programming language that is translated into machine code before execution, as opposed to interpreted languages. Compiled languages, such as C, C++, and Go, offer high performance and efficiency.

Complex

In programming, complex numbers are numbers with real and imaginary parts, used in scientific and engineering applications. Complex data types are supported in languages like Python, MATLAB, and R.

Component

A component is a modular part of a hardware or software system, designed to perform a specific function. Components enable reuse, scalability, and maintainability in system design.

Computational Biology

Computational biology is the application of data analysis, mathematical modeling, and simulation to study biological systems. It supports research in genomics, proteomics, and systems biology.

Computational Chemistry

Computational chemistry uses computer simulation to solve chemical problems, predict molecular properties, and design new materials. It combines chemistry, physics, and computer science.

Computational Complexity Theory

Computational complexity theory classifies computational problems by their inherent difficulty and resource requirements. It defines complexity classes such as P, NP, and NP-complete, guiding algorithm design and analysis.

Computational Model

A computational model is a mathematical representation of a system or process, used to simulate and analyze complex phenomena. Computational models are essential in science, engineering, and economics.

Computational Neuroscience

Computational neuroscience applies mathematical models and computer simulations to understand brain function, development, and cognition. It bridges neuroscience, psychology, and computer science.

Computational Physics

Computational physics uses numerical analysis and algorithms to solve physical problems, simulate experiments, and analyze data. It is a key tool in research, engineering, and education.

Computational Science

Computational science is an interdisciplinary field that uses computing to solve complex problems in science, engineering, and society. It involves modeling, simulation, and analysis of systems using high-performance computing.

Computational Steering

Computational steering is the practice of manually intervening in an autonomous computational process to change its outcome. It enables interactive exploration, optimization, and control of simulations.

Computer Case

A computer case is the enclosure that houses the main components of a computer, including the motherboard, power supply, drives, and cooling systems. Cases come in various sizes and form factors, affecting expandability and airflow.

Computer Fan

A computer fan is an active cooling device that circulates air inside or around a computer case, preventing overheating of components. Fans are essential for maintaining system stability and performance.

Computer Form Factor

Computer form factor refers to the physical dimensions, layout, and mounting of hardware components, such as motherboards, cases, and power supplies. Common form factors include ATX, microATX, and Mini-ITX.

Computer Graphics

Computer graphics is the field of creating, manipulating, and displaying images and animations using computers. It encompasses 2D and 3D graphics, rendering, modeling, and visualization.

Concatenation

Concatenation is the process of joining two or more strings, arrays, or data structures end-to-end. Concatenation is widely used in programming, data processing, and text manipulation.

Concurrency

Concurrency is the ability of a system to execute multiple tasks or processes simultaneously, improving efficiency and responsiveness. Concurrency is achieved through threads, processes, and asynchronous programming.

Conditional

A conditional is a programming construct that performs different actions based on whether a Boolean condition evaluates to true or false. Conditionals enable decision-making and control flow in software.

Configuration

Configuration refers to the setup and customization of hardware or software options to meet specific requirements. Configuration files, settings, and tools enable flexible and adaptable systems.

Connection

A connection is a link between hardware components, software modules, or networked devices, enabling communication and data exchange. Connections can be physical (cables, ports) or logical (protocols, APIs).

Connector

A connector is a port, plug, or interface used to join hardware components or devices. Connectors enable modularity, expandability, and interoperability in electronic systems.

Console

A console is a text-based interface or terminal that allows users to interact with a computer system, run commands, and monitor output. Consoles are used for system administration, development, and troubleshooting.

Container

A container is a lightweight, portable package that includes everything needed to run software consistently across different environments. Containers isolate applications, dependencies, and configurations, enabling reproducible deployments and scalability.

Control Store

The control store is the memory that stores the microcode of a CPU, defining how instructions are executed at the hardware level. Control stores enable flexible and upgradable processor designs.

Conventional Peripheral Component Interconnect (Conventional PCI)

Conventional PCI is a computer bus standard for attaching hardware devices, such as network cards, sound cards, and storage controllers, to a motherboard. PCI provides high-speed data transfer and expandability.

Corrective Maintenance

Corrective maintenance is the process of fixing errors or defects in software or hardware after deployment. It involves identifying, diagnosing, and resolving issues to restore normal operation.

Core

A core is the part of a CPU that performs arithmetic and logical operations. Modern processors contain multiple cores, enabling parallel processing and improved performance for multitasking and complex workloads.

Core Memory

Core memory, originally referring to magnetic core memory, is now used as a synonym for main memory (RAM) in modern computers. Core memory stores data and instructions for active processes.

CRAN

CRAN (Comprehensive R Archive Network) is the primary repository for R packages, enabling users to download, install, and share extensions for the R programming language. CRAN supports reproducible research and data analysis in statistics and science.
