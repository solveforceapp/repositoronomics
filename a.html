A Terms

Absolute Path

An absolute path is the complete address or location of a file or directory within a file system, starting from the root directory. It specifies every directory and subdirectory traversed to reach the file, ensuring there is no ambiguity regardless of the current working directory. For example, in Unix-like systems, /home/user/documents/file.txt is an absolute path, while in Windows, C:\Users\Name\Documents\file.txt is an absolute path. Absolute paths are essential for scripting, automation, and system administration, as they provide a reliable way to reference files and directories across different environments.

Accelerated Graphics Port (AGP)

The Accelerated Graphics Port (AGP) is a high-speed point-to-point channel designed for attaching a video card to a computer’s motherboard, primarily to assist in the acceleration of 3D computer graphics. Introduced by Intel in 1997, AGP provided a direct connection between the graphics card and the system memory, allowing for faster rendering of complex images and textures compared to the older PCI standard. AGP slots were common on motherboards throughout the late 1990s and early 2000s, but have since been replaced by PCI Express (PCIe) in modern systems.

Accelerator

An accelerator is a specialized hardware device, such as a microprocessor, ASIC (Application-Specific Integrated Circuit), or expansion card, designed to offload specific computational tasks from the main CPU. Examples include graphics accelerators (GPUs), AI accelerators for machine learning, and cryptographic accelerators for secure data processing. By handling dedicated workloads, accelerators improve overall system performance and efficiency, especially in high-demand applications like gaming, scientific computing, and artificial intelligence.

Accumulator

The accumulator is a register within a computer’s arithmetic logic unit (ALU) that temporarily stores intermediate results of arithmetic and logic operations. It is a fundamental component in early computer architectures and is still conceptually present in modern CPUs. The accumulator allows for efficient chaining of operations, as the result of one calculation can be immediately used as input for the next, reducing the need for repeated memory access.

Adaptive Maintenance

Adaptive maintenance refers to modifications made to software after its initial deployment to ensure it continues to function correctly as its environment changes. This may include updates for compatibility with new operating systems, hardware, or third-party software. Adaptive maintenance is a key aspect of the software lifecycle, ensuring longevity and usability as technology evolves.

Address Space

Address space is the range of memory addresses that a process or device can use to access data. In computing, address space can refer to physical memory (RAM), virtual memory, or memory-mapped devices. Each process typically has its own address space, providing isolation and security. The size of the address space is determined by the architecture (e.g., 32-bit or 64-bit), affecting how much memory can be directly accessed.

Advanced Technology eXtended (ATX)

Advanced Technology eXtended (ATX) is a motherboard and power supply configuration specification developed by Intel in 1995. ATX improved upon previous standards by introducing better power management, more efficient airflow, and standardized mounting points for components. The ATX form factor remains widely used in desktop computers, with variations such as microATX and Mini-ITX for smaller systems.

AI Accelerator

An AI accelerator is a type of hardware designed specifically to speed up artificial intelligence and machine learning tasks, such as neural network training and inference. These accelerators, which include GPUs, TPUs (Tensor Processing Units), and custom ASICs, are optimized for the parallel processing and matrix operations common in AI workloads. They are used in data centers, edge devices, and consumer electronics to enable faster and more efficient AI applications.

Algebraic Data Type (ADT)

An Algebraic Data Type (ADT) is a composite type used in computer science and programming languages, particularly functional languages. ADTs are defined by combining other types using operations like sum (union) and product (tuple), allowing for the creation of complex data structures. Examples include Option or Either types in Scala and Haskell, which help model data that can take on multiple forms.

All-in-One Computer

An all-in-one computer integrates the monitor and computer components into a single unit, reducing cable clutter and saving space. These systems are popular in offices and homes for their streamlined design and ease of setup, though they may offer less upgradability compared to traditional desktop PCs.

Alpha Testing

Alpha testing is an early phase of software testing performed by developers or internal staff before releasing the product to external users (beta testers). The goal is to identify bugs, usability issues, and design flaws in a controlled environment, allowing for fixes before broader testing or public release.

AMOLED

Active-Matrix Organic Light-Emitting Diode (AMOLED) is a display technology used in smartphones, tablets, and TVs. AMOLED screens offer vibrant colors, deep blacks, and fast response times by using organic compounds that emit light when an electric current is applied. The active-matrix design allows for precise control of individual pixels, improving image quality and power efficiency.

Analog-to-Digital Converter (ADC)

An Analog-to-Digital Converter (ADC) is an electronic device that converts continuous analog signals (such as sound or light) into digital data that computers can process. ADCs are essential in audio recording, digital cameras, and sensor systems, enabling the integration of real-world data into digital systems.

Analytical Engine

The Analytical Engine was a proposed mechanical general-purpose computer designed by Charles Babbage in the 19th century. Although never completed, it introduced key concepts such as a central processing unit, memory, and programmable instructions, laying the groundwork for modern computing.

Antistatic Mat

An antistatic mat is a safety device used to prevent electrostatic discharge (ESD) when working with sensitive electronic components. By grounding the user and equipment, antistatic mats protect hardware from damage during assembly, repair, or maintenance.

Antikythera Mechanism

The Antikythera Mechanism is an ancient Greek analog computer dating back to around 100 BCE. Discovered in a shipwreck, it was used to predict astronomical positions and eclipses, demonstrating advanced engineering and mathematical knowledge for its time.

Application-Specific Integrated Circuit (ASIC)

An Application-Specific Integrated Circuit (ASIC) is a custom-designed chip created for a particular application or function, such as cryptocurrency mining, networking, or audio processing. ASICs offer high performance and efficiency for their intended tasks but lack the flexibility of general-purpose processors.

Argument

In programming, an argument is a value or reference passed to a function, method, or command-line program. Arguments allow functions to operate on different data and enable code reuse. For example, in print("Hello, World!"), "Hello, World!" is an argument to the print function.

Arithmetic Logic Unit (ALU)

The Arithmetic Logic Unit (ALU) is a core component of the CPU responsible for performing arithmetic operations (addition, subtraction, multiplication, division) and logic operations (AND, OR, NOT, XOR). The ALU processes data and instructions, enabling computers to perform calculations and make decisions.

Arithmetic Operators

Arithmetic operators are symbols used in programming languages to perform mathematical operations on numeric data types. Common operators include + (addition), - (subtraction), * (multiplication), and / (division). These operators are fundamental to data processing and algorithm development.

Array Data Structure (Array)

An array is a data structure consisting of a collection of elements, each identified by an index or key. Arrays store elements in contiguous memory locations, allowing for efficient access and manipulation. They are widely used in programming for tasks such as sorting, searching, and data storage.

Artifact

In software development, an artifact is any tangible by-product produced during the development process. Artifacts include use cases, class diagrams, UML models, requirements documents, design specifications, project plans, and risk assessments. They serve as documentation and reference points throughout the software lifecycle.

Assembly Language

Assembly language is a low-level programming language that uses mnemonic codes to represent machine instructions specific to a computer’s architecture. Assembly provides direct control over hardware, enabling optimization and efficient resource management, but requires detailed knowledge of the underlying system.

Assignment Operator

The assignment operator (=) is used in programming languages to assign a value to a variable. For example, x = 5 assigns the value 5 to the variable x. Assignment operators are fundamental to variable management and program logic.

Associative Array

An associative array, also known as a map or dictionary, is a data structure that stores key-value pairs. Each key is unique and maps to a specific value, allowing for efficient data retrieval, addition, and modification. Associative arrays are widely used in programming for tasks such as configuration management and data indexing.

Attribute

An attribute is a property or characteristic associated with an object, class, or element in programming and data modeling. Attributes define the state or behavior of an object and are used to store data or metadata. For example, a Person class may have attributes such as name, age, and address.

Augmented Reality

Augmented Reality (AR) is a technology that overlays digital information, such as images, sounds, or data, onto the real world through devices like smartphones, tablets, or AR glasses. AR enhances user experiences in gaming, education, navigation, and industrial applications by blending virtual and physical environments.

Automata Theory

Automata theory is a branch of theoretical computer science and mathematics that studies abstract machines (automata) and the computational problems they can solve. It provides the foundation for understanding formal languages, regular expressions, and the design of compilers and interpreters.

Automated Reasoning

Automated reasoning is a field of computer science and mathematical logic focused on developing algorithms and software that enable computers to reason, prove theorems, and solve logical problems automatically. Applications include formal verification, artificial intelligence, and knowledge representation.
