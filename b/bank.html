Bank

Definition

In computer memory, a bank refers to a group of memory modules that can be accessed simultaneously, allowing for parallel data transfer and improved performance. Memory banks are used in RAM and cache architectures to increase bandwidth and efficiency.

Application

Banks are essential in high-performance computing, enabling simultaneous access to multiple memory modules. This architecture is used in modern RAM, graphics cards, and server systems to optimize data throughput and reduce latency.
